Plugin per la gestione di immobili per agenzie immobiliari e imprese edili

== Description ==

ENG
Plugin for the management of products compatible with woocommerce slug. Allows the management of categories, colors, sizes, technologies and features.

ITA
Plugin per la gestione dei prodotti compatibile con gli slug di woocommerce. Permette la gestione di categorie, colori, taglie, tecnologie e caratteristiche.

### Url del plugin all'indirizzo http://www.rswstudio.it/wordpress/plugin-prodotti-per-wordpress/

### Plugin prodotti by http://www.rswstudio.it

== Installation ==

= ENG =
1. Upload 'prodotti' directory  to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= ITA =
1. Carica la cartella 'prodotti'  via ftp su '/wp-content/plugins/'
2. Attiva il plugin attraverso la sezione 'Plugins' dentro all'area riservata WordPress
